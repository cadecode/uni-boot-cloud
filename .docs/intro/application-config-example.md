### gateway 服务配置

uni-boot-gateway-dev.yml（DEV 环境）

```yaml
server:
  port: 8000

spring:
  cloud:
    gateway:
      routes:
        - id: uni-boot-framework
          uri: lb://uni-boot-framework
          predicates:
            - Path=/framework/**
          filters:
            - StripPrefix=1
        - id: uni-boot-example
          uri: lb://uni-boot-example
          predicates:
            - Path=/example/**
          filters:
            - StripPrefix=1
```
